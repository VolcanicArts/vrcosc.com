## App
- Added support for the system button on VR controllers
    - The system button click can be reserved by SteamVR and may not function
    - Only Knuckles and Quest controllers have support currently
- Fixed some Windows 11 compatibility issues with the file picker
- Fixed mute toggle and mute set not working for users with less than 66 FPS

## Pulse
- Added Controller System node for the system button on VR controllers
- Fixed Delay node not continuing when the delay was less than 1

## Modules
- Fixed ClientInfo not working in the ChatBox
- Improved PiShock data gathering and logging
    - PiShock will now get all the initial values of the parameters if available
    - More logging has been added to inform about what it's trying to do
