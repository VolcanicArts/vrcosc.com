## App
- Added a setting to allow users to change the OSCQuery target client name
    - This is for people who want to use VRCOSC with ChilloutVR, as they support the same standard of parameters that VRChat does
    - This is not an indication of official ChilloutVR support
- Fixed PiShockProvider sometimes having "multiple shockers with the same sharecode" error
- Added VRChat's desktop FPS and exposed the VRChatClient class to the SDK
- Fixed deleting a ChatBox clip not removing it from the UI
- Added a filter to the run screen's avatar parameter tab
- Fixed clip variable dragging for people on Windows 11

## Pulse
- Changed FireIfX cancellation behaviour to only cancel the flow if the condition is met
- Fixed PhysBoneParameterSource node not working for float parameters
- Added Minimum and Maximum numeric nodes
- Collapsed various nodes
- Fixed DictionaryKeyToValue node throwing an exception if the key isn't found
- Fixed DictionaryAdd node throwing an exception if adding failed
- Fixed SoundPlayer type defaulting to the system library
- Fixed SpeechSource node not updating trigger nodes
- Fixed value-only nodes updating when they shouldn't be
- Fixed UserCameraSetAperture node name

## Modules
- Optimise PiShock's calls and guarded against several errors
- Fixed PiShock's account URL being changed and leading to a 404 page
- Added ability for a Counter instance to be reset when the day rolls over instead of when the modules restart
- Added FPS to ClientInfo
- Fixed instance count ChatBox variable not updating in ClientInfo
- Added KAT module
    - This allows users to send text using KillFrenzyAvatarText
    - This is only available in Pulse
